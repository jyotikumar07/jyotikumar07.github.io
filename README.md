# Tech-Forward Bookkeeping & Financial Audit Case Study

## Project Overview
This project simulates an end-to-end month-end closing process for a modern consulting firm (Apex Tech Consulting). It demonstrates cloud-compatible ledger management, bank feed ingestion, automated matching workflows, and a successful financial audit using QuickBooks Online Accountant.

## Core Competencies Demonstrated
* **Accounts Receivable Architecture:** Pre-recorded and tracked client invoices totaling $4,900.00 for major accounts (Acme Corp, Global Tech, Main St Retail).
* **Bank Feed Ingestion & Mapping:** Cleared a 25-row complex transaction ledger, cleanly segmenting multi-vendor operational expenses (Utilities, Travel, Software/SaaS).
* **Internal Control Audit:** Successfully investigated, calculated, and resolved an initial $440.00 balance discrepancy by auditing net cash change ($6,152.96) against statement targets.

## Final Deliverables & Proof of Work
* **Reconciliation Variance:** $0.00 (Flawless Month-End Close)
