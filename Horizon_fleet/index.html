Horizon Fleet Logistics - Financial Summary Dashboard

An interactive, executive-facing Power BI dashboard engineered to monitor cash flows, reconcile financial ledgers, and dynamic category tracking.

📌 Technical Project Highlights

  • Data Cleaning & Transformation: Utilized Power Query to clean transaction ledgers, isolate operational expenses, and build a unified star schema data model.
  • Financial Reconciliation: Developed key metrics (Service Revenue, COGS, Net Profit) via DAX, ensuring 100% mathematical alignment back to the cash flow ledger.
  • Advanced DAX Mechanics: Resolved multi-way category ties by writing a custom TOPN dynamic table and EARLIER filter-context measure to automatically group smaller expenses under a balanced "Other" slice.

📈 Business Action & Key Insights

  • Identify Expense Leakage: Instantly surfaced a $1.2K engine repair outlier as the month's largest expense, enabling rapid cost-auditing.
  • Working Capital Monitoring: Tracked daily cash inflow vs. outflow via an area trend chart, identifying key squeeze points throughout the month.
